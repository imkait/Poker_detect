# Poker_detect
Poker detect with yolov8

自訓練Yolov8n撲克牌模型  
若撲克牌張數為5張   
會進行撲克牌牌型偵測  

1. 下載
git clone https://github.com/imkait/Poker_detect  
或  
下載ZIP

2. 安裝套件
pip install -r requirements.txt
